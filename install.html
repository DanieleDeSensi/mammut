Get it

If you only need to monitor the local machine Mammut has no external dependencies. If remote monitoring is needed, Google's Protocol Buffer Library should already be installed on your machine.


Quick-start Instructions

~ $ git clone git://github.com/DanieleDeSensi/mammut.git

~ $ cd Mammut

~/Mammut $ make

~/Mammut $ make install
